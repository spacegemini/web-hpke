# web-hpke - ECC/HPKE 密钥管理与加解密工具

[English](./README.md) | 简体中文

基于浏览器本地运行的 **ECC/HPKE (P-256)** 密钥生成与文件加解密工具。安全、离线、无需上传私钥，为您提供极致的隐私环境。

## 🌟 核心特性

- **💯 纯本地运行 (Client-Side Only)**：所有加解密运算均在您的浏览器内完成，没有任何数据会上传至服务器。
- **🔒 高度安全与离线支持**：支持在完全断网的情况下使用！私钥仅保存在本地，最大程度防范集中式数据泄露风险。
- **⚡ ECC/HPKE P-256 算法**：采用目前前沿的安全标准，提供强健的非对称加密。
- **📁 文件与文本全支持**：可以加密任何类型的文件（生成 `.hpke` 后缀文件），同时也支持快速加密普通文本消息。
- **🌍 开源透明**：零黑盒，代码完全开源。用户可以随时通过浏览器的开发者工具查看执行逻辑。
- **📱 PWA 桌面/移动端支持**：可将工具作为应用程序直接添加到系统桌面或手机主屏，享受原生应用体验。

## 📌 快速入门指南

### 1️⃣ 生成密钥
打开工具后，点击 **"生成新密钥对"**。您会得到：
- **私钥 (Private Key)**：用于解密。**绝对保密，切勿发给他人！丢失则无法找回数据！**
- **公钥 (Public Key)**：用于加密。可以公开发送给需要给你发文件或消息的人。

### 2️⃣ 加密文件 / 文本
如果你想发送私密信息/文件给对方：
1. 索要对方的 **公钥** 并在页面对应位置填入。
2. 选择要加密的文件，或输入要加密的文本消息。
3. 点击加密，系统会生成 `.hpke` 加密文件或 Base64 密文。直接把文件/密文发送给对方即可。

### 3️⃣ 解密文件 / 文本
如果你收到了被加密的内容：
1. 在解密区域填入你自己的 **私钥**。
2. 上传收到的 `.hpke` 加密文件或粘贴收到的文本密文。
3. 点击解密，即可直接获取/下载原始信息和文件。

